# PYENV & PYENV-VIRTUALENV CHEAT SHEET Ubuntu WSL2 Pyenv Installer: https://github.com/zaemiel/ubuntu-pyenv-installer ## CHECK PYENV INSTALLATION pyenv --version ## INSTALLING PYTHON VERSIONS pyenv install --list # Show available versions pyenv install 3.12.2 # Install Python 3.12.2 pyenv uninstall 3.12.2 # Remove installed version ## USING PYTHON VERSIONS pyenv global 3.12.2 # Set global Python version pyenv local 3.11.0 # Set local version for a project pyenv shell 3.9.1 # Temporarily use a Python version ## CREATING VIRTUAL ENVIRONMENTS pyenv virtualenv 3.12.2 myenv # Create virtual environment pyenv virtualenvs # List virtual environments pyenv activate myenv # Activate environment pyenv deactivate # Deactivate environment pyenv uninstall myenv # Remove virtual environment ## USING VIRTUAL ENVIRONMENTS LOCALLY pyenv local myenv # Set virtual env for a project python -m venv venv # Alternative (without pyenv) ## UPDATING & CLEANUP pyenv update # Update pyenv & plugins pyenv rehash # Refresh shims ## TROUBLESHOOTING pyenv doctor # Check for common issues pyenv which python # Locate Python binary in use
